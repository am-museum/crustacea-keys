DELTA data files and images

Downloading the data files

The data zipfile contains the DELTA data and directives files, the HTML and RTF descriptions, the character and taxon images, documentation files such as the Introduction, and various other subsidiary files.

After downloading the file, unzip it in a folder 'sample\' reserved for this dataset, e.g. ‘\mydeltadata\sample\’.

See also Guidelines for using data taken from Web publications.

Downloading the images

Note. The images for the sample data don't need to be downloaded separately, as they are included in the data zipfile (see above). This section is included only to exemplify how the images can be downloaded separately if they have been omitted from the data zipfile, in order to reduce its size.

The images for a particular character or taxon can be viewed directly from the interactive key or the descriptions. There is no need to download the images unless you want to use them offline.

The images are listed on the image links page.

To download the images, run the DELTA program Getimages in a command window in the ‘sample\’ folder (see above). The images will be downloaded to a subfolder ‘sample\images\’.

Getimages uses the free program Wget, which is not included in the DELTA distribution. For information about Wget, see ‘Auxiliary programs’ in Installing and running the programs of the DELTA System.
